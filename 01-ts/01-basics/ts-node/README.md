# ts-node

`ts-node` is a tool that allows you to run TypeScript files directly without manual compilation.

- **Install:**
  - Use npm: `npm install -g ts-node`
- **Usage:**
  - Run TypeScript files: `ts-node <filename>.ts`
- **Benefits:**
  - Faster development workflow, no need to compile before running.

`ts-node` is ideal for scripts, prototyping, and development environments.
